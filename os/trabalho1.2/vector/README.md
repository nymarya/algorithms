# Vector

## About

The class Vector consists of implementing the abstract data type (ADT) list, using structural properties and operations. 

## How to run

| #       | Description           | Command  |
| :------------- |:-------------| :-----|
| 1      | Compile | ```$ arm-linux-gnueabihf-g++ -std=c++11 -c src/drive_vector.cpp -I include -o vector``` |
| 2      | Run   | ```$ ./vector``` |
